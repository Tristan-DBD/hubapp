## 🌟 Pourquoi HubApp ?

Vous en avez marre des applications qui vous traquent, qui nécessitent Internet pour s'ouvrir, ou qui vous obligent à installer dix outils différents pour des tâches simples ?

HubApp est différent.

**C'est une boîte à outils modulaire qui tourne entièrement sur votre appareil.** Pas de compte obligatoire (sauf si vous voulez synchroniser Google Drive). Aucune donnée ne quitte votre téléphone sans votre consentement. Juste un lanceur élégant, sombre et rapide qui évolue avec vos besoins.

## 📱 Que pouvez-vous faire avec ?

| Outil | À quoi ça sert |
|---|---|
| **💰 Finances personnelles** | Suivez vos revenus et dépenses, archivez les anciens relevés, passez d'un mois à l'autre. Toutes vos données restent sur votre téléphone. |
| **🏋️ Coach Upload** | Sélectionnez des vidéos de coaching, nommez-les et envoyez-les directement sur Google Drive. Organisation intelligente des dossiers incluse. |
| **📁 File Upload** | Choisissez n'importe quel fichier ou média, compressez-le si vous voulez, et expédiez-le vers Drive. Parfait pour les sauvegardes et le partage. |

Et parce que HubApp est modulaire, **d'autres outils peuvent être ajoutés** sans gonfler ceux que vous utilisez déjà. Chaque fonctionnalité est totalement indépendante.

## 🔥 Ce qui rend HubApp unique

- **📴 100% offline par défaut** — Pas d'Internet ? Aucun problème. HubApp fonctionne intégralement hors ligne. Les fonctions Drive ne s'activent que si vous les utilisez.
- **🔒 Confidentialité avant tout** — Zéro télémétrie. Zéro analytics. Zéro appel réseau en arrière-plan. Vos données vous appartiennent.
- **⚡ Ultra rapide** — Les modules se chargent uniquement quand vous les ouvrez. L'écran d'accueil se lance instantanément, à chaque fois.
- **🎨 Sombre et élégant** — Thème noir et violet, agréable pour les yeux et magnifique sur les écrans AMOLED.
- **🧩 Ajoutez ce dont vous avez besoin** — Chaque outil est un module. Utilisez uniquement ce qui vous intéresse, ignorez le reste.
- **☁️ Google Drive à la demande** — Importez des fichiers, sauvegardez vos données, parcourez vos dossiers. Fonctionne aussi hors ligne — vos imports en attente ne seront pas perdus.

## 🛠️ Pour les développeurs

HubApp est open source (licence MIT). Construit avec React Native.

```sh
npm start                 # Lancer le serveur de dev
npm run android           # Installer la version debug
npm test                  # Lancer les tests
npm run release:publish   # Builder & publier sur GitHub
```
